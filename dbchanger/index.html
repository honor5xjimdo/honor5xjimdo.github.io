底包更换系列 [原创禁止转载！]

前言

先上链接：百度云盘 微云盘

WARNING:不允许以任何形式转载或外传！

适用：emui4，rom随意，官方官改非官方都可以

作用：

  降级包：快速降级到418/9，推荐EMUI4用户使用（EMUI3切勿使用！！！）

  升级包：快速升级到437/441，用于刷437/441官改

(请深刻理解这些卡刷包的作用，刷入会帮你升级/降级底层版本)

 

注意：如果是刷机的话，必须先刷rom再刷速改！！！！

 

注意：底包为443刷速改会上锁！！！(开机出现 boot image is very faild!就是已经上锁)（切记保存好解锁码！！）开机前要在REC下先重启到fastboot解锁解锁一次！（后续继续刷机无需解锁）

用法

  1. 下载对应卡刷包
  2. 刷入
  3. ok已经回退完成

注意（这里是重点）

降级包：

由于你刷完后已经降级到了419，但是之前你的手机还能开机意味着回到419之后就无法开机了（无限重启），所以你需要：：

如果是emui用户：

  • 刷入一个第三方rom

非emui用户：

你之前应该是刷底包补丁刷入的当前rom，那么

既然现在是419了，不需要底包补丁，去工具区下载ab补丁取消掉ba补丁的效果

 

升级包：

刷完后已经升级到了437/441，意味着现在无法开机了（无限重启），所以你需要去工具区下载Ba补丁修复

 

PS：ba补丁即大多数人说的底包补丁，otg补丁，修复底包之类的东西

须知：如果你不是全网通或者dm用户，你会发现网盘内没有对应的卡刷包。

是的，本次只共享全网通和全网通DM版的包，其他包请加群下载

或

在捐赠区捐赠>=0.5RMB后捐赠留言处留下邮箱和你需要的东西
